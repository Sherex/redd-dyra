# Redd Dyra App
A web app for Redd Dyra Telemark which keeps track of all cats, tasks, daily logs, shifts and more.

## Development
```bash
git clone https://github.com/Sherex/redd-dyra

cd redd-dyra

# Backend + frontend
npm run dev
# Backend
npm run dev:b
# Frontend
npm run dev:f
```
